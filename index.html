About SoSketchy

This is a personal project of Lorenz Yboa for the application of knowledge and skills in web development and web application. This web application is inspired by toys for children called "Itch-a-sketch", which I remember playing with my friends and having fun drawing what-ever we want.

SoSketchy is made of pure HTML, CSS, and Javascript. No libraries, frameworks, etc. But my future projects will include the tools and latest technology in web develpment. Hope you enjoy SoSketchy and I am open for more coffee at my table at any time.
